# 🔴 Minha Pokédex - Aprendendo do Zero

Este é um projeto de estudo criado com o objetivo de **aprender React do zero na prática**. O foco principal desta aplicação é entender o funcionamento dos Hooks e como realizar o consumo de uma API externa mantendo o código limpo e organizado.

## 🎯 O que eu aprendi e pratiquei neste projeto

- **Fundamentos do React:** Estruturação de componentes e JSX.
- **Hooks (useState e useEffect):** Gerenciamento de estado da aplicação e controle do ciclo de vida/efeitos colaterais.
- **Consumo de API REST:** Utilização do `fetch` nativo do JavaScript para buscar dados na [PokeAPI](https://pokeapi.co/).
- **Renderização Condicional:** Criação de estados de carregamento (a famosa tela de "Procurando Pokémons no mato alto...").
- **Manipulação de Arrays:** Uso do `.map()` para renderizar listas dinâmicas de componentes no front-end.

## 🚀 Tecnologias Utilizadas

- React
- JavaScript (ES6+)
- HTML5
- Tailwind CSS (para uma prototipação rápida e responsiva)
